# TwitterWordPhrequency

A python project used to provide sendiment analysis on Twitter targets.. I mean accounts.

## Quickstart

```
pip install -r requirements.txt
export PATH=${HOME}/.local/bin:$PATH
./TwitterWordPhrequency.py
```
